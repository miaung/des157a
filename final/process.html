For my final project, I have chosen to focus on visual design. With my visual design, I am trying to recreate the feeling of old Pokemon games, specifically with battling other Pokemon. The image above shows the general visual design of what I plan to achieve, but I will make it more pixelated. I added more buttons, meaning I'll probably have to edit the code to add more moves. I added these buttons because one button didn't feel like a fighting game, and I thought that it would make the game more interesting. I also plan to add more drop shadows so that you can see the content better, and I also want to add a hover over the buttons.
